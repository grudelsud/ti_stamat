Search Images

below a working example

/* From URL */
{
"index":"split",
"source":"url",
"fileidentifier":"http:\/\/cdn.hypebeast.com\/image\/2012\/07\/bandit9-nero-motorcycle-1.jpg",
"feature":"featureCEDD",
"numberofresults":10
}
/* From Index */
{
"index":"split",
"source":"index",
"fileidentifier":"84845",
"feature":"featureCEDD",
"numberofresults":10
}

index must be the name of an existing index

source must be either index or url

fileidentifier can be a URL or file descriptor as saved in the index, according to what selected for source

feature must be one of:

featureAutoColorCorrelogram, descriptorScalableColor, featureCEDD, featureColorHistogram, descriptorColorLayout, featureTAMURA, descriptorEdgeHistogram, featureFCTH, featureGabor, featureJCD, featureJpegCoeffs, featureSift, featureSurf
